# Voice and Phone Controlled RC Car

An Arduino-based RC car that can be controlled using voice commands through a Bluetooth-enabled smartphone. The project integrates Arduino UNO, HC-05 Bluetooth module, motor driver, ultrasonic sensor, and servo motor for smooth movement and obstacle detection.

## Features
- Voice & Bluetooth control
- Obstacle detection
- Arduino UNO based
- Easy to build and use

## Components
- Arduino UNO
- HC-05 Bluetooth Module
- L293D Motor Driver
- Ultrasonic Sensor
- Servo Motor
- DC Motors
- Battery Pack

## Technologies Used
- Arduino IDE
- Embedded C/C++
- Bluetooth Communication
